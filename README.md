## About the project:

### Topic
	Movies and TV Series tracking Website.
### Database entities

	- User:
	
		- Anonymous.
		- Registered User.
		- Administrator.
		
	- Movie.
	- TV Series.
	- Genres
	- Viewed.
	- Report notifications.
	- Comments.
	
![diagrama bbdd.png](https://github.com/SergioSV96/kingdomTV/blob/master/KingdomTV_ScreenShots/diagrama bbdd.PNG)	
	
### Advanced functionality

	- Uses Database TMDb (The Movie Database).
	- Logging with e-mails.
	
	
### Graphic information
	We are going to use pie charts showing the genre that users like the most.
	
### Other functionality
	
	- Send e-mails.
	- Report spam and report spoiler comments.
